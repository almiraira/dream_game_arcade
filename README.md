# Игра "Ловец снов"

Атмосферная 2D-игра на Python + Arcade. Управляй сияющей душой, собирай осколки воспоминаний и добирайся до Ловца снов сквозь лабиринты ночных кошмаров.

---

## Установка и запуск

```bash
pip install -r requirements.txt
python main.py
```

> Требуется Python 3.11+


---
## Управление

| Клавиша | Действие |
|---------|----------|
| W / ↑  | Лететь вверх |
| S / ↓  | Лететь вниз |
| A / ←  | Лететь влево |
| D / →  | Лететь вправо |
| Escape  | Главное меню |
| Enter   | Подтвердить (в меню/финале) |

---


## Геймплей

- Собирай **звёзды** (осколки воспоминаний) — каждая даёт 10 очков.
- Избегай **кошмаров** — тёмных туч, которые патрулируют уровень. При столкновении теряется шкала «Осознанности».
- Долети до **Ловца снов** — это конец уровня.
- Если «Осознанность» упадёт до нуля — сон прерывается.
